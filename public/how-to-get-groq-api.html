Setup Guide

How to Get a Groq API Key

Free tier, no credit card required  ·  Takes less than 2 minutes

1

Go to the Groq Console

Open your browser and navigate to console.groq.com. This is Groq's official developer dashboard. No payment info needed to sign up.

2

Create an Account

Sign up using your Google account, GitHub account, or any email address. Groq's free tier gives you immediate access to fast inference on models like Llama 3 and Mixtral.

3

Open API Keys

Once you're in the dashboard, look for "API Keys" in the top-right navigation menu. Click it to open the key management page.

4

Generate a New Key

Click the "Generate Key" button. A dialog will appear asking you to configure your new key.

5

Name Your Key & Set Expiration

Enter a name for the key. Anything works, e.g. "pygrind". Set the expiration date to your preference (or leave it at the default). Then click Submit.

6

Copy Your Key. One Time Only

Groq will display your API key exactly once. Copy it immediately and save it somewhere secure (e.g. a password manager or a local notes file). Once you close the dialog, the key cannot be retrieved again — you would need to generate a new one.

Once copied, paste it into the PyGrind setup field. That's it — you're ready to start drilling.

Security Notice

PyGrind never stores your API key on any server. Your key is saved exclusively in your browser's local storage and is sent directly from your device to Groq's servers. PyGrind has no backend, no database, and no proxy — it is impossible for us to see or log your key. We recommend creating a dedicated key for PyGrind so you can revoke it independently if needed.
