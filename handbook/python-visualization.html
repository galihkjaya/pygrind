Visualization: Matplotlib & Seaborn

Turn data into visual explanations, not decorative confetti.

1. Overview

Visualization helps inspect distributions, compare groups, detect outliers, understand relationships, and communicate results. A metric says what happened; a good plot often explains why.

Mental model

Every chart maps data to visual features: position, length, color, shape, and size. Choose the mapping that answers one question clearly. One plot, one message. Jangan jadi dashboard pasar malam.

What you should be able to do after this page

Choose appropriate chart types, build Matplotlib and Seaborn plots, customize labels, and avoid misleading visualizations.

3. Hands-on Code

Matplotlib object-oriented API

Use fig, ax because it scales better than random pyplot calls.

import matplotlib.pyplot as plt

months = ["Jan", "Feb", "Mar", "Apr"]
sales = [120, 150, 90, 180]

fig, ax = plt.subplots(figsize=(7, 4))
ax.plot(months, sales, marker="o")
ax.set_title("Monthly Sales")
ax.set_xlabel("Month")
ax.set_ylabel("Sales")
ax.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

Distribution and outliers

Histograms show shape. Boxplots summarize median, spread, and outliers.

values = [52, 55, 56, 60, 62, 64, 65, 66, 90]

fig, ax = plt.subplots(figsize=(7, 4))
ax.hist(values, bins=6)
ax.set_title("Score Distribution")
ax.set_xlabel("Score")
ax.set_ylabel("Frequency")
plt.show()

Seaborn with DataFrames

Seaborn works nicely with Pandas DataFrames and statistical plots.

import seaborn as sns
import matplotlib.pyplot as plt

tips = sns.load_dataset("tips")
ax = sns.scatterplot(data=tips, x="total_bill", y="tip", hue="time")
ax.set_title("Tip vs Total Bill")
plt.tight_layout()
plt.show()

Correlation heatmap

Heatmaps are useful for scanning relationships, but not proof of causality.

corr = tips[["total_bill", "tip", "size"]].corr()
sns.heatmap(corr, annot=True, vmin=-1, vmax=1)
plt.title("Correlation Matrix")
plt.show()

4. Practical Workflow

Good visualization starts from the analytical question, not from “chart apa yang aesthetic?” Pick chart type based on variable type and task.

  1. State the question in one sentence.
  2. Identify variable types: numeric, categorical, time, text, geospatial.
  3. Choose chart type: distribution, relationship, comparison, composition, or trend.
  4. Plot the simplest version first.
  5. Add title, axis labels, units, and readable formatting.
  6. Remove decoration that does not help interpretation.

5. Common Mistakes

Traps to avoid
  • Using pie charts for too many categories. Your reader is not a human protractor.
  • Truncating axes to exaggerate differences.
  • Using 3D charts when 2D is clearer.
  • Plotting too many categories without sorting or grouping.
  • Treating correlation heatmaps as causal evidence.
