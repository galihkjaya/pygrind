1. Overview

A neural network is a stack of differentiable transformations trained to reduce a loss. Building a tiny network from scratch makes PyTorch and TensorFlow feel less magical and more like organized matrix math.

Mental model

Each layer transforms inputs into new representations. Forward pass makes predictions, loss measures error, backprop computes gradients, and optimization updates weights. Repeat until the model becomes less wrong.

What you should be able to do after this page

Understand neurons, weights, bias, activation functions, loss, gradients, backprop intuition, and a minimal training loop with NumPy.

3. Hands-on Code

One neuron forward pass

A neuron is a tiny scoring function.

import numpy as np

x = np.array([0.5, 1.2, -0.7])
w = np.array([0.8, -0.4, 0.3])
b = 0.1

z = x @ w + b
a = max(0, z)  # ReLU
print(z, a)

Tiny regression training loop

This is the skeleton hiding inside bigger libraries.

import numpy as np

X = np.array([[1.0], [2.0], [3.0], [4.0]])
y = np.array([[2.0], [4.0], [6.0], [8.0]])

w = np.random.randn(1, 1)
b = np.zeros((1,))
lr = 0.01

for epoch in range(500):
    pred = X @ w + b
    error = pred - y
    loss = np.mean(error ** 2)

    grad_w = (2 / len(X)) * X.T @ error
    grad_b = (2 / len(X)) * error.sum(axis=0)

    w -= lr * grad_w
    b -= lr * grad_b

print(w, b, loss)

Softmax intuition

Softmax converts logits into probabilities that sum to 1.

logits = np.array([2.0, 1.0, 0.1])
exp = np.exp(logits - logits.max())
probs = exp / exp.sum()
print(probs)

4. Practical Workflow

When learning deep learning, do not start with a huge model. Start with a tiny network and make sure loss decreases.

  1. Define input and output shapes.
  2. Initialize parameters.
  3. Run forward pass.
  4. Compute loss.
  5. Compute gradients.
  6. Update parameters with learning rate.
  7. Track loss over epochs and inspect predictions.

5. Common Mistakes

Traps to avoid
  • Ignoring tensor shapes. Shape mismatch is the deep learning jump scare.
  • Using too large a learning rate and making loss explode.
  • Forgetting nonlinear activations so stacked layers collapse into one linear model.
  • Not normalizing inputs.
  • Judging training only by final accuracy without checking loss curves.
